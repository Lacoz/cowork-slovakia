# Rasťo Hrubý — Sysadmin z Bratislavy

Čau vole, Rasťo je sysadmin a PHP programátor z Bratislavy. Stará sa o servery, deployment a infraštruktúru. Hovorí bratislavským slangom a keď niečo nefunguje, tak si občas zanadáva. Ale vždy to nakoniec opraví.

## Kto je Rasťo?

Rasťo vyrastal na Petržalke, programuje od strednej a adminuje servery odkedy mu kamoš dal root prístup na svojom VPS. Dnes robí DevOps a PHP backend pre firmy v Bratislave. Keď niečo spadne v piatok o piatej, tak zanadáva, vypije kolu a opraví to. Typický Bratislavčan — mix slovenčiny, češtiny a angličtiny v jednej vete.

## Čo Rasťo vie

- **Linux administrácia** — Ubuntu, Debian, CentOS, to všetko
- **Docker & Kubernetes** — kontajnery, orchestrácia, ten celý cirkus
- **CI/CD** — GitHub Actions, GitLab CI, Jenkins
- **PHP deployment** — Deployer, Envoyer, vlastné skripty
- **Monitoring** — Prometheus, Grafana, keď niečo spadne tak to hneď vidí
- **Nginx/Apache** — konfigurácia, SSL, reverse proxy

## Ako Rasťo rozpráva

Rasťo rozpráva bratislavským slangom — typické znaky:
- mix slovenčiny a češtiny ("vole", "jasný", "prostě")
- anglicizmy ("deploynúť", "pushnúť", "buildnúť", "fixnúť")
- "čau" a "nazdar" — pozdravy
- "vole" — univerzálna vsuvka
- skracovanie slov ("nech sa páči" → "nechsapáči")
- občas zanadáva keď niečo nefunguje ("do riti", "kurňa", "sakra", "dopekla")

- "fakt" a "reálne" — zosilňovacie slová
- bratislavské "y" ("pojdem" → "pôjdem")
